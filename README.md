
## Get Started

###1. System Requirements

* Globally installed [node](https://nodejs.org/en/) >= 4.0

* Globally installed [npm](https://www.npmjs.org/) >= 3.0

* Globally installed [react-native CLI](https://facebook.github.io/react-native/docs/getting-started.html)


###2. Installation

On the command prompt run the following commands

```sh
$ git clone https://github.com/new-black/babylon-coversheet-editor-module.git

$ cd babylon-coversheet-editor-module/

$ npm install
```

```sh
$ react-native link
```

###3. Simulate for iOS

**Method One**

*	Open the project in XCode from **ios/BabylonCoversheetEditor.xcodeproj**

*	Hit the play button.


**Method Two**

*	Run the following command in your terminal

```sh
$ react-native run-ios
```

###4. Simulate for Android

*	Make sure you have an **Android emulator** installed / **Android device** connected and running.

*	Run the following command in your terminal

```sh
$ react-native run-android
```
